系统要求

  • 操作系统：
    ✨ 优先推荐: Ubuntu 18.04+ 或 Debian 10+（兼容性最佳）
    同样支持: CentOS 7+、RHEL 7+、Fedora 30+
  • 架构： x86_64 (AMD64)
  • 内存： 至少 512MB RAM
  • 磁盘空间： 至少 100MB 可用空间
  • 权限： Root 或 sudo 访问权限

💡 专业建议： 我们推荐使用 Ubuntu 20.04 LTS 或 Debian 11 以获得最佳体验和稳定性。
